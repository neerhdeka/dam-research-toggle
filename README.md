# DAM Research Toggle

Static HTML prototypes for DAM (Digital Asset Management) research concepts.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
# Python 3
python3 -m http.server 8000

# Then open http://localhost:8000
```

## Run online (GitHub Pages)

1. Push this repo to GitHub (already done if you cloned from [dam-research-toggle](https://github.com/neerhdeka/dam-research-toggle)).
2. In the repo: **Settings** → **Pages**.
3. Under **Source**, choose **Deploy from a branch**.
4. Branch: **main**, folder: **/ (root)**. Click **Save**.
5. After a minute or two, the site will be live at:
   **https://neerhdeka.github.io/dam-research-toggle/**

## Concepts

- **Concept 2** – Dropdown

Start from `index.html` (or the root URL on GitHub Pages) to pick a concept.
